# Stickman Mesh Generator (MVP)

## Что это
Процедурный генератор **единого** lowpoly-меша стикмена в **T-позе** без дыр, собираемый экструдом граней (голова → шея → грудь/живот/таз → руки/ноги).

Файлы:
- `StickmanMeshGenerator.cs` — компонент генерации (runtime)
- `Editor/StickmanMeshGeneratorEditor.cs` — кнопка `Generate Mesh` в инспекторе (editor-only)
- `QuadShellMeshBuilder.cs` — минимальный quad-shell экструдер (flat normals)

## Как использовать в Unity
1. В сцене создайте `GameObject` (например `Stickman`).
2. Добавьте компонент `StickmanMeshGenerator`.
   - `MeshFilter` и `MeshRenderer` добавятся автоматически.
3. Назначьте материал в `MeshRenderer` (любой).
4. В инспекторе нажмите **Generate Mesh**.
## Параметры
- **Head / Neck**: размеры куба головы, длина/масштаб шеи.
- **Torso**: длины 3 секторов и их масштаб (меняет “толщину” капа при экструде).
- **Arms / Legs**: длины сегментов; `capScale` — насколько сужается/расширяется поперечник при экструде.

## Примечания MVP
- Нормали — **flat** (вершины дублируются на грань), UV/vertex colors отсутствуют.
- Меш генерируется в **локальных** координатах объекта; “пол” примерно около `Y=0`.
- Если хотите строже контролировать точку выхода рук (не через поиск граней), следующий шаг — хранить `FaceId` нужных боковин прямо в коде генерации (или расширить builder для возврата боковых граней при `Extrude`).
