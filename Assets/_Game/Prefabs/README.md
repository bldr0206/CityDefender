# Prefab Workspace

Recommended prefab groups:

- `Layouts`
- `Towers`
- `Enemies`
- `UI`

Layout prefabs should always include `LevelLayoutAuthoring`.

Optional: add `LevelLayoutBoundsGizmo` on the **layout root** to edit play-area bounds in the Inspector (`Local Center` / `Local Size`); a wire box is drawn in the Scene view (not visible in Game view).
